📌 IPL Match Prediction & Analytics System

This project builds a complete machine learning pipeline to analyze and predict IPL match outcomes using historical data.

🚀 Features
Match outcome prediction using ML models
Feature engineering with:
Team strength
Recent performance (rolling window)
Venue bias
Toss impact

Model comparison:
Logistic Regression
Random Forest
XGBoost

Additional modules:
Venue recommendation system
Player of the Match prediction (KNN)
Toss decision strategy (Decision Tree)
Fan sentiment analysis (lexicon-based)

📊 Results
Evaluated using:
ROC-AUC
Precision-Recall
Feature importance

🛠️ Tech Stack
Python
Pandas, NumPy
Scikit-learn
XGBoost
Matplotlib

📂 Project Workflow
Data → Feature Engineering → Modeling → Evaluation → Insights
