# Employee Salary Segmentation using Unsupervised Clustering

## Overview

This project analyzes employee salary data from Scaler and applies unsupervised machine learning techniques to identify meaningful employee segments based on compensation, experience, and job characteristics.

## Dataset

- 193K+ employee records
- Company information
- Job positions
- Salary (CTC)
- Experience

## Techniques Used

- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis
- K-Means Clustering
- Hierarchical Clustering
- PCA
- Elbow Method

## Tools

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Key Insights

- Identified salary-based employee segments
- Analyzed compensation patterns across experience levels
- Generated business recommendations for career growth and hiring strategies
